이 폴더에 실제 사진을 넣으세요.

권장 파일 예시
- hero.jpg          : 메인 상단 큰 사진 (가로 긴 형태, 1200x600 권장)
- ok20.jpg          : OK20 방제드론 사진
- agras.jpg         : DJI AGRAS 사진
- xag.jpg           : XAG P100 사진

사진을 넣은 뒤, 관리자 페이지(admin.html)에서
해당 항목의 "사진 경로" 칸에 images/파일명.jpg 형태로 입력하면 됩니다.

경로를 비워두거나 파일이 없으면 자동으로 이모지/플레이스홀더가 표시됩니다.
